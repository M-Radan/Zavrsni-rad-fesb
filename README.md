# Aplikacija za pomoć osobama s poremećajem prepoznavanja boja

**Naslov na engleskom jeziku:**  
**Application for Assisting People with Color Vision Deficiency**

---

## Sažetak

Cilj ovog rada bio je izraditi mobilnu aplikaciju za pomoć osobama s problemom prepoznavanja boja, poznatim kao daltonizam. Aplikacija je razvijena korištenjem **Android Studio** platforme, programskog jezika **Java** i biblioteke **OpenCV**.  

Korisnicima je omogućeno:
- Jednostavno odabiranje originalne i zamjenske boje  
- Automatska zamjena prepoznate originalne boje odabranom bojom  
- Pohrana podataka o bojama u lokalnu memoriju mobilnog uređaja  

Provedeno testiranje pokazalo je da aplikacija učinkovito prepoznaje i zamjenjuje definirane boje, čime olakšava svakodnevne aktivnosti osobama s daltonizmom.

---

## Abstract (English)

The goal of this project was to develop a mobile application to assist individuals with colour recognition difficulties, commonly known as colour blindness. The application was created using the **Android Studio** platform, the **Java** programming language, and the **OpenCV** library.  

Users can:
- Easily select an original and a replacement colour  
- Automatically replace the detected original colour with the chosen one  
- Store colour data in the local memory of the device  

Testing has shown that the application effectively detects and replaces the defined colours, thereby facilitating everyday activities for individuals with colour vision deficiency.

## Zadatak rada

U završnom radu student će izraditi aplikaciju za mobilne uređaje koja pomaže osobama s poremećajem prepoznavanja boja.  

**Funkcionalnosti aplikacije:**
- Dohvaćanje slike s ugrađene kamere  
- Prikaz slike u realnom vremenu s promijenjenim vrijednostima onih slikovnih elemenata čija nijansa spada u definirane intervale  
- Intervali boja koje se mijenjaju postavljaju se u konfiguracijskoj datoteci aplikacije preko odgovarajućeg sučelja  

**Za razvoj aplikacije koristi se:**  
- Android Studio
